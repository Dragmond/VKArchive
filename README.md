# VKArchive

VKArchive — приложение для экспорта переписок VK в автономный HTML-архив.

## Возможности

- Экспорт личных сообщений и бесед
- Красивый HTML-интерфейс
- Поддержка:
  - фотографий
  - видео
  - документов
  - голосовых сообщений
  - аудио
  - стикеров
  - ссылок
  - опросов
  - геометок
  - записей на стене
  - подарков
  - граффити
  - историй
  - товаров
  - подборок товаров
  - звонков
- Ответы на сообщения
- Пересланные сообщения
- Системные сообщения
- Отображение пользователей и сообществ
- Автоматическая загрузка медиафайлов
- Пакетное сохранение сообщений в SQLite
